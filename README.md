# Movie Data API

A RESTful backend API built with Python, Flask, and SQLite to manage and query movie data. Supports CRUD operations and dynamic filtering via URL query parameters.

---

## Features

- Retrieve all movies or filter by rating, year, or title
- Add new movies
- Update movie ratings
- Delete movies
- Returns JSON responses for easy integration with front-end or analytics

---
